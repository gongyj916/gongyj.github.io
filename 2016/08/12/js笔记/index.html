substring和substr的用法

substring

用于提取字符串中介于两个指定下标之间的字符
返回一个新的字符串，该字符串值包含 stringObject 的一个子字符串，其内容是从 start 处到 stop-1 处的所有字符，其长度为 stop 减 start。

substring 方法返回的子串包括 start 处的字符，但不包括 end 处的字符。
如果 start 与 end 相等，那么该方法返回的就是一个空串（即长度为 0 的字符串）。
如果 start 比 end 大，那么该方法在提取子串之前会先交换这两个参数。
如果 start 或 end 为负数，那么它将被替换为 0。

substr

用于返回一个从指定位置开始的指定长度的子字符串。
substr(start [, length ]):
start 必需。所需的子字符串的起始位置。字符串中的第一个字符的索引为 0。
length 可选。在返回的子字符串中应包括的字符个数。
如果start为负数，则start=str.length+start。
如果 length 为 0 或负数，将返回一个空字符串。
如果没有指定该参数，则子字符串将延续到stringObject的最后。

获取控制

. 任意字符
o* 表示匹配0个或多个o
o+ 表示匹配1个或多个o
o? 表示匹配0个或1个o
{m,n} o{2,4} 表示匹配2个到4个o
[a-z0-9] 匹配小写a-z，0-9都行
[^a-z0-9] 匹配非小写a-z，且非0-9其他都行
^[a-z0-9] 匹配从行首开始第一个字符为小写or数字
[a-z0-9]$ 匹配从行尾第一个字符为小写or数字
\w 匹配a-z，A-Z及都行
\W 匹配非a-z，A-Z及都行
\d 匹配0-9都行
\D 匹配非0-9都行
