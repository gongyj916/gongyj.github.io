css 长度单位

em：1em = 离自己最近元素大小的1倍
2em =离自己最近元素大小的2倍

rem：浏览器根节点有个默认值：比如html{font-size = 16px}，rem等于html默认值的备数。

vw：视宽，比如8vm，手机屏幕宽度200px， 那8vm =（8*200）/100

vh：视高。计算方式同vh

css 字体

font-family ：字体是单个单词的不需要引号；字体类型名称由多个单词组成，需要用“ ”括起来。

@font-face{ //自定义字体类型
font-face: 自定义字体名称;
src:url(字体文件路径);
}
